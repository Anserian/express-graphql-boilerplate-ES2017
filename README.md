# express-graphql-boilerplate

## Features

- Express server
- GraphQL
- GraphQl tools
- eslint with airbnb
- winston logger

## Installation

```
npm install
```

```
yarn install
```

## Usage

To run production
```
npm start
```

```
yarn start
```
The app will be serving from http://localhost:4000/graphiql/

To run dev
```
NODE_ENV=dev npm start
```

```
NODE_ENV=dev yarn start
```

GraphiQL will be running at http://localhost:4000/graphiql/
